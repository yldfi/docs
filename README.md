# yldfi

yldfi provides automated yield vaults on Ethereum built on [Yearn V3 yVault technology](https://docs.yearn.fi/getting-started/products/yvaults/v3), following the ERC-4626 standard. Deposit your tokens and earn auto-compounding returns without manual intervention.

## Available Vaults

| Vault | Token | Strategy | Fee | LlamaLend Collateral |
|-------|-------|----------|-----|----------------------|
| [ycvxCRV](vaults/ycvxcrv.md) | cvxCRV | Convex auto-compounder | 20% (15% LlamaLend + 5% yldfi) | Yes |
| [yscvxCRV](vaults/yscvxcrv.md) | cvxCRV | Convex auto-compounder | 5% (yldfi) | No |
